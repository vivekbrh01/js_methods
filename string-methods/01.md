# JavaScript String Exercises

### is_string
Write a JavaScript function to check whether an `input` is a *string or not*.

Example:
```js
  // your code goes here
  console.log(is_string('tony stark'));
  true
  console.log(is_string([1, 2, 4, 0]));
  false
```
